
To use:

- Install 'ReST Editor' from Eclipse market place
- Install sphinx from http://pypi.python.org/pypi/Sphinx (you may need to install 'easy_install' first)
- Ensure that the 'make' command is in your path.
- Edit the PATH argument in the run configuration 'sphinx.user.doc'.
- Run the run configuration 'sphinx.user.doc' provided. 
- Refresh the build folder to see the output.
